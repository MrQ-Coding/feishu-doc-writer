# feishu-doc-writer

Codex skill for drafting and rewriting structured Chinese Feishu technical notes with compact, source-backed explanations.

## Install

### macOS / Linux

One-command install:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/MrQ-Coding/feishu-doc-writer/main/scripts/install.sh)
```

From a local clone:

```bash
bash scripts/install.sh
```

### Windows PowerShell

One-command install:

```powershell
irm https://raw.githubusercontent.com/MrQ-Coding/feishu-doc-writer/main/scripts/install.ps1 | iex
```

From a local clone:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\install.ps1
```

### Windows Git Bash

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/MrQ-Coding/feishu-doc-writer/main/scripts/install.sh)
```

## Notes

- Default install path: `~/.codex/skills/feishu-doc-writer`
- Set `CODEX_HOME` first if you use a non-default Codex home directory.
- Restart Codex after installing or updating the skill.
